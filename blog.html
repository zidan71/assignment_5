What is the Document Object Model (DOM)?

The Document Object Model (DOM) is a programming interface for web documents.
It shows the structure of an HTML document as a tree of nodes, where each node corresponds
to part of the document (like elements, attributes, or text).

How do you select an element from the DOM?

You can select an element from the DOM using JavaScript by using methods like,
document.getElementById('id') - selects an element by its ID.
document.getElementsByClassName('class') - selects elements by their class name.
document.querySelector('selector') - selects the first element matching a CSS selector.
document.querySelectorAll('selector') - selects all elements matching a CSS selector.

What is event delegation in the context of the DOM, and why is it useful?

Event delegation is a technique in the DOM where you attach a single event listener to a parent
element instead of multiple listeners to individual child elements. When an event occurs,
it bubbles up the DOM, and the parent element can handle the event for its children.
It's useful because you can reduce the number of event listeners, which improves performance.
Dynamic Content: It works even for elements added to the page later.

How do you manipulate an element's attributes and styles using the DOM?

For attributes: We can use setAttribute() to change an attribute
and getAttribute() is used to read an attribute. removeAttribute() can be used to delete an attribute.
also use the .style property to apply css styles
